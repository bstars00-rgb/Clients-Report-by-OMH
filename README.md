# OhMyHotel — Weekly Top-Selling Hotels

A public, partner-facing platform showing **weekly top-selling hotel rankings** by country.

- **Data basis**: Booking date, 1-week window (Sat–Fri)
- **Coverage**: Japan (Top 100), Korea (Top 50), Vietnam (Top 50)
- **Shows**: rank, week-over-week rank change, hotel name, city
- **Excludes**: all confidential commercial data (revenue, margin, channel, TTV)
- **Features**: country / city / movement filters, search, EN·KO, dark/light mode

Same data is shared with all partners — one consistent public view, refreshed weekly.
